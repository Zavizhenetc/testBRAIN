1.Клонируем 

``git clone https://github.com/Zavizhenetc/test4brain.git``

2.переходим в папку с проектом 

``` cd test4brain```

3.устанавливаем зависимости 

``npm i `` 

4.Собираем  

``npm run build``

5.Разворачиваем на локальном  

``npm run dev``
